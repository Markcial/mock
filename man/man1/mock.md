mock(1) -- Mock library for testing
============================================

## SYNOPSIS

mock COMMAND [EXIT_CODE] [FUNCTION_BODY] [--help]<br>

## USAGE
## DESCRIPTION
## OPTIONS

COMMAND the command willing to mock

EXIT_CODE defaults to 0

FUNCTION_BODY if empty no code will be executed

* -h, --help:
    Show usage help.

## EXAMPLES

```fish
mock ls 0
```
